## 三、给美工的通用注意（**反复踩坑，务必看**）

1. **透明底必须是真 alpha 透明**。这几批反复出现「把假透明背景导成实色像素」：
   - 骰子：透明区导成了**洋红**（chroma-key 色）；
   - 商品卡：导成了**灰白棋盘格**；
   - 进步卡面：边距导成了**部分不透明的棋盘格**。
   开发这边是靠脚本把这些背景抠掉才能用，费时且偶有毛边。**请直接导出真透明 PNG，不要带任何键色 / 棋盘格 / 实色背景。**

2. **进步卡竖卡请统一比例**。现在交付的卡有 256×341、也有 256×384，比例不一会导致「文字叠进卡面文字框」时位置飘。请所有进步卡 / 发展卡用**同一画布尺寸与文字框位置**。

3. **玩家棋子（村庄 / 城市 / 路 / 骑士）一律交白模**，主体白 / 浅灰、无固有色——引擎按玩家色实时染色。别上色、别导键色底。
