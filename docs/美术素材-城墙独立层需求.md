# 美术需求 · 城墙单独成层

> 承接建筑白模那批。只解决一个问题：**带城墙的城市 / 大都会，城墙要能一眼看出来**。

## 问题

带城墙的白模（`piece-city-walled-white`、`piece-metropolis-walled-white`）里，**城墙和建筑是同一种白**。引擎在运行时按玩家颜色对整块模型染色（白 → 红/蓝/白/橙），于是**城墙也被染成了玩家色**，和建筑融成一体，看不出这座城有没有城墙。

（临时办法：我在程序里把模型底部一圈保持白色来近似城墙——能用但不够干净，所以还是想请你出一版正规的。）

## 方案：城墙单独导一层，不参与染色

把**城墙单独导成一个透明图层**（石灰色、不染色），引擎把它叠在「已染成玩家色的建筑」上面。这样城墙**恒为石色**、和玩家色建筑对比鲜明，一眼可辨。

## 交付物

**首选（2 张）** —— 城墙层对齐现有的建筑白模：

| 文件 | 内容 |
|------|------|
| `piece-city-wall-key.png` | 只有城墙那一圈护墙（石灰色、透明底），画布尺寸与摆放位置**和 `piece-city-white` 完全一致** |
| `piece-metropolis-wall-key.png` | 大都会的城墙层，对齐 `piece-metropolis-white` |

> 前提：带城墙模型里的**建筑部分**和不带城墙的 `piece-city-white` / `piece-metropolis-white` **位置一致**（这样直接叠加即可）。

**备选（4 张）** —— 如果带城墙的建筑与不带城墙的摆位不同，就把带城墙模型**拆成两层**（互相对齐）：
`piece-city-walled-base-white`（建筑，白模可染）+ `piece-city-walled-wall`（城墙，石色不染）；大都会同理。

## 规格

- **城墙用中性石灰色**（暖灰，约 `#cbc2b0`），要求：在红/蓝/白/橙四色建筑上都清晰可辨，且不像任何一个玩家色。
- **与建筑层像素级对齐**：同一画布尺寸（1024² 源图）、同一摆放位置，叠上去严丝合缝。
- 建筑层（白模）保持**纯白**、可被引擎染色；城墙层**不参与染色**。
- **真 alpha 透明**：除城墙外全透明。**务必别把透明背景导成洋红 / 棋盘格 / 实色**（骰子、商品、进步卡都栽在这上面，我这边得写脚本抠掉，费时且有毛边）。

## 接入方式（交付后我来做）

带城墙城市 = `染色(piece-city-white)` + 叠加 `piece-city-wall`（不染）。大都会同理。当前程序里的「底部留白」到货即替换。
